@GwtCompatible
public interface ClassToInstanceMap<B>
extends Map<Class<? extends B>,B>

A map, each entry of which maps a Java raw type to an instance of that type. In addition to implementing Map, the additional type-safe operations putInstance(java.lang.Class, T) and getInstance(java.lang.Class) are available.

Like any other Map<Class, Object>, this map may contain entries for primitive types, and a primitive type and its corresponding wrapper type may map to different values.
