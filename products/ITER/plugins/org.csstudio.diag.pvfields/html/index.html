Overview

The PV Fields viewer is a diagnostic tool that displays information about a Process Variable.

By default it displays information obtained by connecting to the life control system channel, which includes the name of the host that serves the channel, the underlying data type etc.

Site-specific extensions can include information from for example an Irmis-type relational database that has information about the original value of fields, the record type and more.

Usage

Enter a PV name as suggested by the tool tip. The tool will show information for that record.
