Saving Changes

After editing some values, the changes can be written to the underlying process variables by selecting File/Save from the menu bar. Alternatively, the toolbar icon or a keyboard shortcut like Ctrl-S may be available.

When closing a PACE table with modified values, you will also be asked if you want to save the changes or close without saving.

Before actually writing the changes to the PVs, a dialog as shown below will prompt for information to be entered into the ELog.

  • Enter your ELog user name and password.
  • You can edit the title and text, which starts out as a list of modified PVs and their values
  • Press OK to perform the ELog entry and PV update.
  • .. or press Cancel.

Save not available

The option to save changes is only available when there are any modified values. This is also indicated via an asterix in the PACE table title. If no values where modified from the PVs' original values, there is nothing to save and the option to do so is not available.
