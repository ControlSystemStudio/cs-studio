CSS caSnooper

The CSS caSnooper is a simple program to display statistics of broadcasts on the network.

To start using the program you have to choose the display interval from combo box and press button "start snooping".
This will print new results in table and statistics in statistics box everytime the interval is timed. While snooping, the green circle will appear next to the combo box.

To stop the snooping you simply click button "stop snooping".
Red icon will appear where green one was, this indicates that snooper is in stop mode.

The CSS caSnooper table can also be filled with remote data from Directory view.
To fill it, you need to find the casnooper instance in Directory view, right click on the instance and click on Get Table. In order to do this, caSnooper must be in stop mode(red icon shown)
