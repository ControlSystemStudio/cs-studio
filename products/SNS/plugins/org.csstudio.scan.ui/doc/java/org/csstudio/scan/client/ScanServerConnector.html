public class ScanServerConnector

Connect to a ScanServer

Example for connecting to scan server: 

  ScanServer server = ScanServerConnector.connect();
  ...
  ... server.getScanInfos();
  ...
  ScanServerConnector.disconnect(server);

Default host and port to which to connect are defined in the ScanServer class. For generic Java tools, including Jython, Matlab, ... they can be overriddden via Java system preferences ScanServerHost and ScanServerPort.

Eclipse-based tools set the system properties from Eclipse preferences, so what's used in the end are the system preference values, but they can be configured from Eclipse preferences and the GUI tools include preference pages.
