public class CommandSequence
extends java.lang.Object

Helper for creating a sequence of commands

Used by the client to build a sequence of commands which is then submitted to the server for execution as a "scan".

Allows both List<Command> and plain Command[]. For Java code collections like List are most convenient. Jython nicely converts existing collections into a python list, but it is harder to create collections from a python list, so arrays are also accepted.
