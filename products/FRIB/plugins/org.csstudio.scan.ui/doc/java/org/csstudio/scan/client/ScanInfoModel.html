org.csstudio.scan.client

Class ScanInfoModel

  • java.lang.Object
    • org.csstudio.scan.client.ScanInfoModel

  • public class ScanInfoModel
extends java.lang.Object
    Model of scan information on scan server

    Based on the ScanClient, this model periodically polls the scan server for its state and sends updates to a (GUI) listener.

    Singleton to allow multiple views to monitor the scan server by using a single underlying network connection and poll thread.
