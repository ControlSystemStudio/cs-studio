KEK CSS Changelog

Version numbers in here refer to the release number of KEK CSS.

Version 3.1.2 - 2013-06-05

  • Bug fixes.
  • Fixed BOY to allow execution of Python script with multi-byte characters.
  • Include PyDev update site in "Available Software Sites" by default.
  • Set default character encoding to UTF-8.
  • Abandoned launcher script for cERL Office and PF-AR Office. To monitor cERL and PF-AR, use the launcher script for PF Office (css_kek_pf_office on Linux and Mac OS X, or css_kek_pf_office.bat on Windows).

Version 3.1.1 - 2012-09-03

  • Based on SNS CSS 3.1.2.
  • Supports CSS launchers for PF, PF-AR and cERL control network environments.
