Overview

The Probe tool allows basic reading and writing of Process Variables (PVs).

Usage

Enter a name into the "PV" text box (followed by 'Return'), and see what happens. The tool should eventually display the current value of the given PV together with time stamp and status.

The 'Adjust' button opens a dialog for writing a new value to the PV.

The 'Save to IOC' button calls the Save Value feature if a plug-in providing this command is installed.

The status bar provides error messages. If all is OK, it displays a slowly averaged update rate of the PV.
