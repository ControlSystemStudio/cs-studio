Quickstart

Um das wiederholte Suchen nach viel genutzen Displays zu vermeiden, gibt es im Navigator das Quickstart-Plugin, welches ein Menu mit Links zu SDS-Dateien im Workspace bereitstellt.

Start displays

Öffnen eines Displays via Quickstart: Klicken auf den SDS-Dateinamen im Quickstart-Menu. Die Datei wird im Run-Modus in einer neuen Shell dargestellt.

Add sds files to quickstart

Das Kontextmenu einer SDS-Datei im Navigator enthält die Aktion 'Add to quickstart'. Diese Aktion fügt die ausgewählte Datei oben in die Dateiliste im Quickstart-Menu ein.

Die Größe der Dateiliste ist auf 20 beschränkt. Wenn man eine weitere Datei hinzufügt, wird die älteste, sprich: die Datei am unteren Ende der Liste, gelöscht

Preferences

Au der Quickstart-Preference-Page kann man mit dem Knopf 'Remove' ausgewählte Dateien löschen oder die Ordnung der Dateiliste mit den Knöpfen 'Up' und 'Down' verändern. Der Knopf 'New' ist abgeschaltet. Um neue Dateien hinzuzugfügen, benutzt man die 'Add to quickstart'-Action im Kontextmenu von SDS-Dateien.
