Quickstart

To avoid searching frequently used displays in the navigator the Quickstart plugin defines a menu in the CSS main menu bar with links to SDS files in the workspace.

Start displays

Open a display via the quickstart just click on the sds file name in the quickstart menu. The file will be shown in the run mode in a new shell.

Add sds files to quickstart

The context menu of a sds file in the navigator defines an action 'Add to quickstart'. This action adds the selected file at the top of the file list in the quickstart menu.

The size of the file list is limited to 20. By adding a new file the oldest file respectively the file at the bottom will be deleted.

Preferences

On the quickstart preference page you can delete the selected file with the button 'Remove' or modify the order of the file list with the buttons 'Up' and 'Down'. The button 'New' is disabled. To add new files please use the 'Add to quickstart' action in the context menu of sds files.
