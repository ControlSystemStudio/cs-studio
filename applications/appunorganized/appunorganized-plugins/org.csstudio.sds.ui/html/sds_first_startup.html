First Startup

SDS Perspective

SDS provides a perspective (a predefined composition of SDS views) for editing displays. It is available under menu 'Window' → 'Open Perspective' → 'Other' → 'Display Development'.
See also the First Startup Flash Demo

To start with a new display it is necessary to create at first a configuration file. Click with the right mouse button in the navigator view → Select 'New' and 'Project'. In the next window select 'Project' and click on 'Next', set the project name and click 'Finish'. Now you can create a SDS configuration file in the new project: Click with the right mouse button on the new project → select 'New' and 'Other'. In the next window select 'Display', Synoptic Display and click on 'Next'. set the display name and click 'Finish'. Now you can open the new display in the editor.
There are some example displays that you can install via menu 'CSS' → 'Display' → 'Install Synoptic Display Studio Sample Displays'.
