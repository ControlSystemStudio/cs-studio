Editor & Widget Palette


Der Editor zeigt die Konfiguration eines Displays im Editor-Modus. Jedes Display beruht auf einer Konfigurationsdatei (Datei-Erweiterung *.css-sds) im Verzeichnis workspace. Im Edit-Modus besteht keine Verbindung zum Kontrollsystem, deswegen ist es nicht möglich, Werte unbeabsichtigt zu ändern. Um ein Display im Ausführungssmodus zu starten, drückt man in der 'Button-Bar' und SDS startet das Display in einer neuen Shell. (Darauf achten, dass die letzten Änderungen aus dem Display-Editor gesichert sind. Ungesicherte Displays sind im Editor tab mit einem '*' vor dem Dateinamen gekennzeichnet).
