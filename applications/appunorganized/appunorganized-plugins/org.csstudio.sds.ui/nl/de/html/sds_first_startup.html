first startup (Erstes Starten)

SDS Perspective

SDS stellt eine Perspektive (eine vordefinierte Zusammenstellung von SDS-Views) zum Editieren von Displays zur Verfügung. Sie ist erreichbar über das Menu 'Window' → 'Open Perspective' → 'Other' → 'Display Development'.
Siehe auch First Startup Flash Demo

Um mit einem neuen Display zu beginnen, muss man zuerst eine Konfigurationsdatei anlegen. Klicken mit der rechten Maustaste in das Navigator View → Auswã§¬en 'New' und 'Project'. Im nächsten Window 'Project' auswählen und auf 'Next' klicken, Projektnamen setzen und 'Finish' anklicken. Nun kann man eine SDS-Konfigurationsdatei im neuen Projekt erzeugen: Rechtsklick auf das neue Projekt → auswählen 'New' und 'Other'. Im nächsten Window auswählen 'Display', Synoptic Display und auf 'Next' klicken. Displaynamen setzten und 'Finish' klicken. Jetzt kann man das neue Display im Editor öffnen.
Es gibt ein paar Beispiel-Displays, die man über Menu 'CSS' → 'Display' → 'Install Synoptic Display Studio Sample Displays' installieren kann.
