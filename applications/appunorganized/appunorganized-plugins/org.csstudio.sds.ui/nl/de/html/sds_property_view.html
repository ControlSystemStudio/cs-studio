Widget Properties

Die Eigenschaften-View zeigt alle Eigenschaften des ausgewählten Widgets im Editor. Wenn mehrere Widgets ausgewählt sind, werden nur die Eigenschaften dargestellt, die allen Widgets gemeinsam sind.
Die folgende Liste enthält nur die Eigenschaften, die zu den meisten Widgets geht. Besondere Eigenschaften findet man im Kapitel 'Synoptic Display Studio Widgets'. Man beachte, dass die Eigenschaften im Display nicht aktualisiert werden, bevor sie gesichert werden.

Dynamisation of properties

SDS unterstützt die Möglichkeit, jede Eigenschaft zu dynamisieren, indem man sie an eine Prozessvariable bindet. Dann steuert der aktuelle Wert der Prozessvariablen das Verhalten der Eigenschaft. Die Eigenschaft verändert sich entsprechend einem Wert, der in Regeln definiert ist.
