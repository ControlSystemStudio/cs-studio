Select, move and resize

Um Widgets auszuw㧬en, benutzt man einfach die Pfeil-Tasten (Cursor-Tasten) auf der Tastatur. Wenn man die Strg-Taste gedrückt hält, wahrend man Pfeil-Tasten betätigt, kann man den Fokus auf ein anderes Widget legen, ohne es auszuwählenen. Dann kann man Strg+Leertaste betätigen, um Widgets individuell auszuwählen. Dieses Feature erlaubt, mit der Tastatur mehrere Widgets auszuwählen.

Um Widgets zu moven und resizen bet㳩gt man die "."-Taste. Damit wird der Editor für aktuell ausgewählte Display in den 'move'-Modus geschaltet. Der 'move'-Modus wird durch ein Indikator-Icon oben im Widget angezeigt (s. Screenshot). Um das Widget auf seine neue Position zu verschieben, betätigt man die Eingabetaste. Man kann auch Esc betätigen, um die 'move'-Operation abzubrechen.

Durch mehrfaches Betätigen der "."-Taste kann man vom 'move'-Modus durch die verschiedenene 'resize'-Modi zyklisch durchgehen. Das Indikator-Icon wird entsprechend erscheinen.
