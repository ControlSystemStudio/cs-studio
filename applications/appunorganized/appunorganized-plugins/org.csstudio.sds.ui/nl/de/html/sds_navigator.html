Synoptic Display Studio

Navigator

Die Navigator-View zeigt die CSS-Projekte im Workspace. Ein CSS-Projekt enthält die Konfigurationsdateien für SDS-Displays und Dateien anderer CSS-Anwendungen.
Um Projekte zu organisieren, kann man (Unter-)Verzeichnisstrukturen erzeugen und 'Linkfolder' verwenden, um externe Quellen mit dem CSS-Workspace zu verbinden. Der CSS Navigator unterstützt CVS1 auf dem Projektlevel.

Linked Folder

Rechtsklicken auf ein Projekt → Auswählen 'New' und 'Other'→. Im nächsten Fenster auswählen 'General' und 'Folder' und klicken auf 'Next' → Klicken auf 'Advanced', auswählen des aus dem Dateisystem in das Projek zu linkenden Verzeichnisses und klicken auf 'Finish'.

*1 Das Concurrent Versions System (CVS), auch bekannt als Concurrent Versioning System, ist ein Open-Source-Versions-Controlsystem, welches die Übersicht über alle Arbeit und alle Änderungen in einer Menge von Dateien, typisch die Implementation eines Software-Projekts, behält und die Zusammenarbeit (möglicherweise weit verstreuter) Entwickler unterstützt.
