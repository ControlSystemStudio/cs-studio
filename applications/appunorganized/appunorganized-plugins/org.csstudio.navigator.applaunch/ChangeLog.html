Application Launcher Changelog

Version numbers in here refer to the plugin org.csstudio.navigator.applaunch.

Version 1.0.2 - 2012-08-29

  • On Mac OS X, native application directories *.app are executed (this was broken after previous update).

Version 1.0.1 - 2012-04-17

  • GANIL added support for scripts with arguments, which are executed via Java Runtime.exec(), not the SWT Program.launch() mechanism.

Version 1.0.0 - 2011-08-10

  • Initial release.
