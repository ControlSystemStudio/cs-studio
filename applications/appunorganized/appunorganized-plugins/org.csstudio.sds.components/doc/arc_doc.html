Arc

This widget displays an arc, i.e. a control that draws an alliptical arc.

Properties

PropertyName in ArcModelIdentifier Description
Transparent BackgroundPROP_TRANSPARENTtransparent_background If true, the background will be transparent.
Start AnglePROP_STARTANGLEstart_angle The start angle of the arc, in degrees.
AnglePROP_ANGLEangle The angle of the arc, in degrees.
FilledPROP_FILLEDfilled If true, the arc will be drawn as a filled elliptical section.
Fill ColorPROP_FILLCOLORcolor.fill The color with which the arc is to be filled.
Line WidthPROP_LINEWIDTHlineWidth Width of the lines the graphic is drawn with.

Implementation

  • the model factory ArcModelFactory

    • Nothing special here, this file is almost the same for all widgets.

  • the model itself, ArcModel

    • This class defines all the properties and provides functions for getting their values (the get* functions).

  • the editpart ArcEditPart

    • This performs initial property synchronisation between the figure and the model (the doCreateFigure function) and registers property listeners. These listeners all just call functions in the figure, which set internal variables.

  • the figure itself, RefreshableArcFigure

    • This is where all the drawing is done. The main class contains functions that outsiders can use to set or get internal variables (the set* and get* functions). The bounding box of the arc defines its ellipse.

    • fillShape draws the background, if not set to transparent and possibly fills the elliptic section.
    • outlineShape draws the arc.
