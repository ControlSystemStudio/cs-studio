Save, Set & Restore

Overview

Save, Set & Restore provides facilities to simultaneously monitor several PVs, store their values, compare the live or stored values with the values at other times, and restore the previously stored values. Application also provides mechanisms to browse and manage the stored sets as well as to export the values to standard file formats.

Getting Started

The two main entities of the Save, Set & Restore application (SSR) are Save Set and Snapshot. Save Set defines the list of PV names, which can be loaded into the spreadsheet, while the Snapshot is a recording of PV values defined by the Save Set at one specific time. Snapshot always belong to a particular Save Set, which is also the way how the entities are made available in the SSR Browser.

The user interface of SSR is composed of three different parts: Save & Restore Browser (view), Snapshot Viewer (Editor), and Save Set Editor (Editor). Browser provides mechanisms to browse and select Save Sets and Snapshots, while the two editors provide facilities, to create or display the contents of the SSR entities.

Browser

Depending on your storage provider the browser is divided into two to four different parts:

  • Branch Selector (Optional) is available in the drop down menu. It displays the list of available branches as provided by the storage provider. A branches can be anything that provides a group of Save sets, snapshots (and base levels if supported).
  • Base Level Browser (Optional) is available as the top most part of the browser view. The base level defines a distinctive set of Save sets, for example for one particular experiment
  • Save Set Browser provides a list of all available Save Sets for the selected branch and base level. If branches and base levels are not supported, the Save Set Browser is the top most selector that is available. The save sets are ordered hierarchically and are displayed in a tree-like view.
  • Snapshot Browser provides a list of all available Snapshots for the selected Save Set. The snapshots are order by time (latest on top) and are displayed together with the creation date, creator name and message/description.

Snapshot Viewer

Snapshot Viewer provides facilities to display the contents of any existing snapshot as well as to take and save new snapshots. The main part of the Snapshot viewer is the table displaying the PVs, their stored and live values. Above the table one can find the common controls (restore, take and save snapshot) an common properties of the snapshot (comment, timestamp, creator and tag). At the very top of the viewer a toolbar is located which provides additional options for viewing the data and exporting/importing data.

The details of the snapshot viewer are described here.

Save Set Editor

Save Set editor allows to create and edit Save set files. Save sets are composed of just two pieces of information: description and pvlist. Editor provides the means to edit these and to save (see the File menu) them by either overwriting the existing save set or into a new file.
