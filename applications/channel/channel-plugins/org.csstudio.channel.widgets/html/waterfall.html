Waterfall

Overview

Waterfall allows to creates a time plot for all the values of a channel query or of a waveform.

Usage

Type in a channel query or a waveform pv. You can customize the plot by right clicking on it. To change the time displayed, you can also drag the range up or down.

The parameters are:

  • Color range: whether the range should be taken from the metadata or from the currently displayed values
  • Scroll: whether the plot should scroll up or down
  • Order by prop: for a channel query, select a property for which to order the channels
  • Resolution: the time period each pixel displays
  • Hide range: whether the range should be displayed or not
