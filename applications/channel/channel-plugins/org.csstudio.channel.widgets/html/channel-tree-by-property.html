Channel Tree by Property

Overview

Channel Tree by Property allows to create an hierarchical view of the channels by using properties and their values. It groups the channels returned by a query based on the value of the properties selected.

Usage

Type in a channel query. Press the button and configure the properties you would like to use. Press apply when you are done. The first level of the tree will display the values for the first property, the second level will display the values for the second property, and so on until all properties are used and you end up with the channel names.

Only channels that actually have values for all properties are going to be displayed. Only property values that are in the query result are going to be diplayed
