Scan System Packages 
Package Description
org.csstudio.scan.client
API for connecting to the scan server.
org.csstudio.scan.command
Scan commands: Set, Loop, ...
org.csstudio.scan.data
Log data types, tools to format logged data.
org.csstudio.scan.device
Device info
org.csstudio.scan.server
Scan server API, used by clients to submit scans, control them, fetch logged data.
Interfaces and classes for user code that interacts with the scan system. For interacting with the scan system from Java, Jython, Matlab, ... code, refer to these:
  • ScanClient
For ScriptCommand scripts that are executed on the scan server with a scan, see:
  • ScanScript
  • ScanScriptContext
For ScanErrorHandlers that handle errors within a scan, see:
  • ScanErrorHandler
  • ScanErrorHandler.Result
