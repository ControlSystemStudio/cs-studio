      • writeXML

        public void writeXML(java.io.PrintStream out,
            int level)
        Write the command (and its sub-commands) in an XML format.

        A command called AbcCommand should write itself as a tag "abc" so that the XMLCommandReader can later determine which class to use for reading the command back from XML.

        Derived classes must call base class implementation to write inherited properties.
