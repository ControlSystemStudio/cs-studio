Logbook
This tool allows entering messages into the logbook:
  1. If your site security policy requires login, you have to enter your user name and password for accessing the logbook
  2. Enter the main text of the entry
  3. Select a logbook (in case more than one is offered)
  4. Optionally, add images (see below)
  5. Press 'Submit'
Images can be added via the "Add Image" button. Depending on the operating system you can also drag-and-drop images from the desktop, the OS X 'finder', or the Windows File Explorer into the gray image area.
