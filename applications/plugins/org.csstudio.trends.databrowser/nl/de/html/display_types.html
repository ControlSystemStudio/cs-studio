Data Display Types

The Config View allows the selection of various "Display Types" for the data.

Area

Dieser Modus stellt die numerischen Werte eines PV als Treppenlinie dar.
Wenn zusätzliche Minimum/Maximum-Information zu diesen Daten verfügbar ist, wird der Minimum/Maximum-Bereich durch eine schattierte Fläche angezeigt.

Min/Max-Info gilt typisch, wenn man den "optimized" Anfragetyp benutzt und ein hinreichend großes Zeitfenster hat. Siehe Request Types.

Boxen auf der Zeitachse zeigen an, dass Daten keine numerischen Werte enthalten. Wenn man drauf klickt, setzt man einen Marker, der anzeigen wird, dass der PV beispielsweise "disconnected" war, wie es im nächsten Bild gezeigt wird. Es kann ähnlichen Boxen in anderen Daten geben. Auch hier gibt ein Marker Unterstützung für Details.

Lines

Dieser Modus stellt die numerischen Werte eines PV als Treppenlinie dar.
Wenn zusätzliche Minimum/Maximum-Information zu diesen Daten verfügbar ist, wird der Minimum/Maximum-Bereich durch zusätzliche Linien angezeigt. Wenn es keine Min/Max-Info gibt, weil gerade Rohdaten dargestellt werden, gilt die Beschreibung im nächsten Modus "Single Line".

Single Line

Dieser Modus stellt die numerischen Werte eines PV als Treppenlinie dar. Es werden keine zusätzlichen Minimum/Maximum-Information angezeigt. Der Modus sollte vermieden werden, weil man Ausreisser in den aktuellen Daten nicht sieht, da nur Mittelwerte dargestellt werden.

Markers

Dieser Modus stellt die numerischen Werte eines PV mit Markern dar. Min/max-Info wird durch Linien angezeigt.

Bars

Dieser Modus stellt die numerischen Werte eines PV als als Balken dar, die immer auf der Zeitachse beginnen. Es werden keine zusätzlichen Minimum/Maximum-Information angezeigt. Dieser Modus ist nur in sehr wenigen Fällen sinnvoll.
