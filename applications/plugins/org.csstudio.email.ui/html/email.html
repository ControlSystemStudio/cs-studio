E-Mail Support

The CSS E-Mail support library allows tools to send electronic mails with optional text and image attachments.

For example, the Data Browser can email a screen-shot of the current plot.

Preferences

There is only one configuration item:
  • SMTP Host:
    Must be set to the name of your SMTP host, i.e. a server that can distribute e-mails.
    When left empty, e-mail support is disabled.
