Search Logbook

This tool allows to search log entries.

Logbook perspective

To search a log entry, type a search query and press enter in the Table Log view. Results are displayed in the table below the query field.
Click on a result to display details in Log Entry view.

Query Search format

Search query format is :
<text> [<keyword>: <value>[, <value>]]

Keywords:
  • logbook: get log entries of given logbooks
  • tag: get log entries with given tags
  • from: get log entries created after given date
  • to: get log entries created before given date
  • properties: get log entries with given properties
Example:
* logbook: Sandbox tag: Design from: lastDay

Note: Autocomplete will help you to write a query.
