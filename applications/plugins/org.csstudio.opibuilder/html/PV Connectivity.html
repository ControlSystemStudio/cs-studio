PV Connectivity

Connect to PV

You have to configure your CSS to connect to your EPICS system before BOY can connect to a real PV. This is configured in the preference page CSS->Preferences->CSS Core->EPICS. For more details, please see here

In BOY, just specify the PV Name property of a PV widget will make the widget connect to the PV automatically in run mode.
