Menu Button

Menu Button will pop up a menu after clicked. The menu will be filled with either the actions from Actions property or the string values from an enumerated PV.

Operations

Click the menu button to popup its menu.

Special Properties

  • Label (label)
    The label text on the button.
  • Actions From PV (actions_from_pv)
    If PV is an enumerated PV, it will automatically fill the menu with actions that will write one of its string values to PV.
  • Transparent (transparent)
    If the background is transparent.
