Knob

A knob widget that can be used to write or read a numeric PV.

Operations

At runtime, dragging the thumb to increase/decrease value in step specified in property Increment.

Special Properties

  • Knob Color (knob_color)
    Color of the knob.
  • Thumb Color (thumb_color)
    Color of the thumb.
  • Increment (increment)
    The least increment/decrement step while dragging the thumb.
  • Show Value Label (show_value_label)
    Show value label on the knob.
