Label

Label is used to display text. It supports multi-line text.

Operations

In OPI editing mode, double click (or click twice) or press F2 while it is selected to edit the text. Press Ctrl+Enter or click other place of the display to finish editing. Press Esc to abort editing.

Special Properties

  • Text (text). The text of the label.
  • Transparent (transparent). Transparent backgournd.
  • Auto Size (auto_size). Automatically adjust the size of the label according to its text.
    Attention: If this is enabled, the size of the widget may subject to change when it startup in Runtime on different platforms. This is because the fonts size on different platforms are different. To guarantee a same or similar looking accross different platforms, it would be better to set this option off and leave enough extra space in the label to allow potential font size change.
  • Horizontal Alignment(horizontal_alignment). The horizontal alignment option for the place of the text in label.
  • Vertical Alignment(vertical_alignment). The vertical alignment option for the place of the text in label.
  • Show Scrollbar (show_scrollbar). Show scrollbar if necessary.
