Polyline

A line or polyline that supports arrows.

Operations

While creating widget, each click will add a new point, double click to finish creating. After created, you are still able to move its points by dragging the yellow square handlers while it is selected. You can flip or rotate it from Orientation context menu.

Special Properties

  • Rotation Angle (rotation_angle). Rotate its points by this angle in degree.
  • Points (points). Points of the polyline.
  • Arrows (arrows). The type of its arrows.
  • Fill Arrow (fill_arrow). If arrow should be filled.
  • Arrow Length (arrow_length). Length of the arrow(s).
