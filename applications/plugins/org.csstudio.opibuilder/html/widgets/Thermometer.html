Thermometer

A widget that indicates numeric value as a thermometer.

Special Properties

  • Fill Color (fill_color)
    Color of the value bar.
  • Color Fillbackground (color_fillbackground)
    Color of the bar background.
  • Show Bulb (show_bulb)
    Show the bulb part of the thermometer.
  • Unit (unit)
    The unit, which could be in celsius, Farenheit, Kelvin or None.
