Check Box

A check box widget which is able to read/write 0/1 to a single bit or the whole PV if Bit property is -1.

Special Properties

  • Bit (bit)
    Which bit of the PV value to read and write. If this is -1 or any negative number, it will read non-zero value as true and zero as false and it will write value 0/1 to the whole PV value.
  • Auto Size (auto_size)
    Automatically adjust the size of the widget.
  • Label (label)
    The text on the label.
