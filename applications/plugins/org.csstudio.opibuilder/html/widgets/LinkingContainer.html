Linking Container

Linking container can be used to embed another OPI or a group in another OPI in it. It is very useful by using it properly. For example, it can be used for animation by loading different content with a rule or script. Combined with different macros, you can show different PVs with same OPI.

Operations

From its context menu, you are able to

  • Reload OPI. Reload its content. (Available to Runtime too.)
  • Perform Auto Size. Automatically adjust its size according to its content.

Special Properties

  • OPI File (opi_file). The OPI file that is linked to.
  • Group Name (group_name). The name of the grouping container in the linked OPI file, which will be loaded if it is specified. If it is not specified, the whole OPI file will be loaded.
  • Zoom to Fit (zoom_to_fit). Zoom its content to fit the size of the container.
