Web Browser

A widget that allows embedding webpage in OPI. The widget uses the native web browser engine from your Operating System, so its context menu may different on different platforms.

Operations

If Show Toolbar property is enabled, there is a browser toolbar to allow you input URL, go forward/backward, stop or refresh.

Special Properties

  • URL (url)
    The URL of the webpage address.
  • Show Toolbar (show_toolbar)
    Show the browser toolbar. This is not changeable at Runtime by rule or script.
