Connection

Connection is a line attatched to two widgets, which will be automatically re-routed when widgets are moved.

Operation

In edit mode, select Connection Tool from palette, attach the connection to two widgets by clicking on two terminals on widgets. If the connection router is Manhattan, you may mannually reroute it by dragging the small yellow square in middle of a line segment. You can also drag connection's terminals to re-attach to other widgets or terminal.

Special Properties

  • Anti Alias (anti_alias)
    If the anti-alias function of the graphics should be on.
  • Arrow Length (arrow_lenght)
    Length of the arrow.
  • Arrows (arrows)
    If there are arrows on two ends of the connection.
  • Fill Arrow (fill_arrow)
    If arrow should be filled.
  • Line Color (line_color)
    Connection color.
  • Line Style (line_style)
    Style of the line.
  • Line Width (line_width)
    Width of the line.
  • Router (router)
    Router type.
