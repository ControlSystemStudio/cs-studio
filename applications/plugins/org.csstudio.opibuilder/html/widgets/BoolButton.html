Boolean Button

A boolean button which is able to write 0/1 to a single bit or the whole PV if Bit property is -1.The direction of the button is automatically determined by the bigger one of its height and width.

Special Properties

  • Square Button (square_button)
    If the button is in rectangle shape.
  • Show LED (show_led)
    Show LED on the button.
