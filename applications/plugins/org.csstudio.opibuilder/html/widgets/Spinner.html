Spinner

A spinner widget that can be used to write and read a numeric PV.

Operations

At runtime, Clicking the arrow buttons to increase/decrease value in step increment. Click text area to edit text. Press ESC key or click other place on the OPI at anytime to abort editing. Press Enter key to submit text. Note: Even if you did not change the text, press Enter will still write the text to PV. It also supports increase/decrease value with keyboard if it gets focused. (It will be focused after clicking on its arrow buttons). Using Up/Down key to increase/decrease value in step increment. Using PageUp/PageDown key to increase/decrease value in page increment.

It has two Runtime configurable properties which can be changed at runtime from context menu Configure Runtime Properties..." : Step Increment and Page Increment.

Special Properties

  • Minimum (minimum)
    Lower limit of the widget.
  • Maximum (maximum)
    Upper limit of the widget.
  • Limits From PV(limits_from_pv)
    Load the Maximum andMinimum limits from PV.
  • Step Increment (step_increment)
    The least increment/decrement step while clicking the arrow buttons or pressing Up/Down key. This is configurable at Runtime from its context menu Configure Runtime Properties....
  • Page Increment (page_increment)
    The increment/decrement step while pressing PageUp/PageDown key.This is configurable at Runtime from its context menu Configure Runtime Properties....
  • Format (format).
    The numeric format for displaying value.
  • Precision (precision).
    Precision of the format. This is meaningful to Decimal and Exponential format.
  • Buttons on Left (buttons_on_left).
    Arrow buttons will be on left side if yes.
