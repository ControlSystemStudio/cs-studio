Widgets

Widgets are the bricks to build an OPI.All the currently available widgets are listed in the Widget Palette. There could be more widgets to be added in future. Benefits from the plug-in mechanism of Eclipse, you may also add your own widgets by implementing a new widgets plug-in. See Creating Customized Widget.

There are four types of widgets:

  • Graphics: The graphics widgets can help you decorate your OPI.
  • Monitors: The widgets that can graphically display the value of PV in various ways.
  • Controls: The widgets by which you can make control operations, for example, write a PV.
  • Others: The other widgets which is not in the three major categories.
  • Layout: Virtual widgets help to automatically layout widgets in a particular way.

The details and examples of all the widgets are available in BOY Examples->10_widgets.opi. Run this OPI see examples of widgets.

Tip: In OPI Editor, press F1 key on Windows, Ctrl+F1 on GTK, Help key on Carbon to open the context help view. It will show related help whenever a widget is selected.
