Class ScanDataIterator

  • java.lang.Object
    • org.csstudio.scan.data.ScanDataIterator

  • public class ScanDataIterator
extends java.lang.Object
    Iterate over ScanData as spreadsheet-type table

    Each column represents the samples for a device. ScanSample serials are used to correlate samples. Each "line" of the spreadsheet contains samples for the same serial. Values for a device are repeated on following lines until a sample with new serial is received.
