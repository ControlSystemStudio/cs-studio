org.csstudio.scan.client

Class ScanClient

  • java.lang.Object
    • org.csstudio.scan.client.ScanClient

  • public class ScanClient
extends java.lang.Object
    Client for remotely accessing the scan server.

    Uses the REST-based web interface of the scan server to submit new scans, monitor and control existing scans.
