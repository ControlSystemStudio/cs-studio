NameSpace Search

Namespace Search (NsS)  will search for a Process Variable in a LDAP structure.

Usage:

NsS will search for a name entered in the white field. While entering a name you can use wildcards:
  • * = no, one or several characters
  • ? = exactly one character
The result is given in a table, where elements diaplayed on the rigth side are on a higher level.
 Elements may be Dragged&Dropped or reused via Mousebutton 3.
