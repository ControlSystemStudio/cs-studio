Name Space Search

Der Namespace Search (NsS)  ermöglicht eine Suche nach Prozessvariablen in einer LDAP Struktur.

Anleitung:

Es wird nach einer Prozessvariablen gesucht. Dabei können Platzhalter in folgender Form verwendet werden:
  • * = keins, eins oder mehrere beliebige Zeichen
  • ? = genau ein beliebiges Zeichen
Das Ergebnis wird tabellarisch dargestellt, wobei das jeweils rechte Element das übergeordnete Element darstellt.
 Alle Elemente können durch Drag&Drop oder per Mousebutton 3 weiter verwendet werden.
