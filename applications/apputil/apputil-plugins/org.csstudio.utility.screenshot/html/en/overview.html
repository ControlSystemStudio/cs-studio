CSS Screenshot

Overview

The screenshot plugin provides a simple capture functionality to get a view or the CSS window as an image. You can then process the image.

The functionalities of the plugin are:

  • Sending the image via email
  • Save image as JPEG or BITMAP
  • Print image (normal or inverted)
  • Adjust the display of the image to the screenshot view(without changing the image data)

Planed functionality are:

  • Using the clipboard of the operating system
  • Snapshot from external windows(also 'Detached Views')

The plugin provides an extension point that allows to define many destination for the image. All destinations are implemented by its own plugin. The size of the image can be changed. The image data will never be changed in case of changing the image size.
