EPICS Library

This provides the ChannelAccess client library for accessing Process Variables on EPICS IOCs.

There is a preference page for basic configuration, where the address list is of primary interest:
Set it to the IP addresses of IOCs or IOC subnets. Refer to the EPICS Channel Access reference manual for details.

Nerdy Details

This uses the 'pure java' version, CAJ. It caused hangups on SNS IOCs, so it is not advisable for production.

See org.csstudio.utility.pv for a utility wrapper.

Note that this includes only the pure Java client code. In addition, one typically also needs the 'caRepeater'. This code does not include the very latest, pure java CA repeater, so you probably have to launch the 'native' version of the repeater for e.g. Win32 yourself before running a CSS app that uses the CA client library.
